# API 参考

## 概述

| 项目 | 值 |
|------|-----|
| 基础 URL | `/api` |
| 数据格式 | JSON |
| 字符编码 | UTF-8 |

## 通用响应格式

### 成功响应

```json
{
  "code": 0,
  "message": "OK",
  "data": { ... }
}
```

### 错误响应

```json
{
  "code": <error_code>,
  "message": "<error_message>",
  "data": null
}
```

## 健康检查

### GET /api/health

检查服务健康状态。

GET /api/health

  "code": 0,
  "message": "OK",
  "data": {
    "status": "healthy",
    "timestamp": "2026-05-02T10:00:00Z"

## 用户管理

### POST /api/users

创建新用户。

**请求体**

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| `username` | string | 是 | 1-50 字符，唯一 | 用户名 |
| `email` | string | 是 | 有效邮箱，1-255 字符，唯一 | 邮箱 |
| `password` | string | 是 | 最小 6 字符 | 密码 |
| `nickname` | string | 否 | 最大 100 字符 | 昵称 |
| `avatar` | string | 否 | 最大 500 字符 | 头像 URL |
| `phone` | string | 否 | 最大 20 字符 | 手机号 |

**响应 (201)**

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "id": 1,
    "username": "alice",
    "email": "alice@example.com",
    "nickname": "Alice",
    "avatar": "",
    "phone": "",
    "is_active": true,
    "created_at": "2026-05-02T10:00:00Z",
    "updated_at": "2026-05-02T10:00:00Z"
  }
}
```

**错误码**

| code | 说明 |
|------|------|
| 1002 | 请求解析错误 |
| 1003 | 参数校验失败 |
| 2002 | 用户已存在 |
| 1301 | 数据库错误 |

### PUT /api/users/:id

更新用户信息。

**路径参数**

| 参数 | 类型 | 说明 |
|------|------|------|
| `id` | int | 用户 ID |

**请求体**（至少传一个字段）

| 字段 | 类型 | 必填 | 约束 | 说明 |
|------|------|------|------|------|
| `email` | string | 否 | 有效邮箱 | 新邮箱 |
| `nickname` | string | 否 | 最大 100 字符 | 新昵称 |
| `avatar` | string | 否 | 最大 500 字符 | 新头像 URL |
| `phone` | string | 否 | 最大 20 字符 | 新手机号 |
| `is_active` | bool | 否 | - | 激活状态 |

**响应 (200)**：同用户信息

**错误码**

| code | 说明 |
|------|------|
| 1002 | 请求解析错误 |
| 1003 | 参数校验失败（至少一个字段） |
| 2001 | 用户不存在 |
| 2004 | 更新用户失败 |

---

### GET /api/users

分页获取用户列表。

**查询参数**

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `page` | int | 1 | 页码 |
| `page_size` | int | 10 | 每页数量 |

**响应 (200)**

```json
{
  "code": 0,
  "message": "OK",
  "data": {
    "list": [
      {
        "id": 1,
        "username": "alice",
        "email": "alice@example.com",
        "nickname": "Alice",
        "avatar": "",
        "phone": "",
        "is_active": true,
        "created_at": "2026-05-02T10:00:00Z",
        "updated_at": "2026-05-02T10:00:00Z"
      }
    ],
    "total": 100,
    "page": 1,
    "page_size": 10
  }
}
```

**错误码**

| code | 说明 |
|------|------|
| 3001 | 无效的页码 |
| 3002 | 无效的每页数量 |

## 完整错误码参考

### 通用错误（1000-1099）

| code | HTTP 状态码 | 说明 |
|------|-------------|------|
| 0 | 200 | 成功 |
| 1000 | 500 | 服务器内部错误 |
| 1001 | 410 | API 已废弃 |
| 1002 | 400 | 请求解析错误 |
| 1003 | 400 | 参数校验失败 |
| 1004 | 400 | 请求转换错误 |
| 1005 | 500 | 数据转换错误 |

### 认证/授权（1100-1199）

| code | HTTP 状态码 | 说明 |
|------|-------------|------|
| 1101 | 401 | Token 无效 |
| 1102 | 401 | Token 解析错误 |
| 1103 | 403 | 权限不足 |
| 1104 | 429 | 超出限流 |

### 文件错误（1200-1299）

| code | HTTP 状态码 | 说明 |
|------|-------------|------|
| 1201 | 404 | 文件获取错误 |

### 数据库错误（1300+）

| code | HTTP 状态码 | 说明 |
|------|-------------|------|
| 1301 | 500 | 数据库错误 |
| 1302 | 500 | 数据库转换错误 |

### 业务错误（2000+）

| code | HTTP 状态码 | 说明 |
|------|-------------|------|
| 2001 | 404 | 用户不存在 |
| 2002 | 409 | 用户已存在 |
| 2003 | 500 | 创建用户失败 |
| 2004 | 500 | 更新用户失败 |
| 2005 | 500 | 删除用户失败 |
| 2006 | 400 | 无效的用户 ID |
| 2007 | 400 | 无效的邮箱格式 |
| 2008 | 400 | 无效的用户名格式 |

### 分页错误（3000+）

| code | HTTP 状态码 | 说明 |
|------|-------------|------|
| 3001 | 400 | 无效的页码 |
| 3002 | 400 | 无效的每页数量 |
